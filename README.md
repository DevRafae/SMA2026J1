# 🌿 Caça-Palavras: Semana do Meio Ambiente M.Cassab 2026

Projeto interativo desenvolvido para engajamento dos colaboradores durante a Semana do Meio Ambiente.

## 🚀 Como Jogar
Basta acessar o link do projeto e encontrar as 7 palavras-chave relacionadas ao equilíbrio natural e sustentabilidade.

## 🛠️ Tecnologias Utilizadas
- HTML5 (Estrutura)
- CSS3 (Identidade visual personalizada)
- JavaScript Vanilla (Lógica do Jogo e Cronômetro)

## 📱 Diferenciais do Projeto
- **Mobile first:** Totalmente otimizado para celulares e tablets.
- **Identidade visual:** Cores e slogans alinhados à campanha da empresa
- **UX protegida:** Bloqueio de seleção de texto e menus contextuais para parecer um aplicativo nativo.
